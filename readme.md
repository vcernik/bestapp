Tato aplikace je skeleton a jako testovací prostředí pro kombinaci webových technologií - DDEV, Nette, Vite, Tailwind, PHPstan


# DDEV
`ddev start`  
`ddev describe`  

Adminer: https://bestapp.ddev.site:9101/

PHPstan: `ddev composer phpstan app --level 4`


# Vite - assets build
`ddev npm install`  
`ddev npm run build`
